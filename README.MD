# cookierun_macro

간단한 `쿠키런 for kakao` 매크로 프로그램

---

## Feature

- 기본적인 인게임 매크로
- 각종 창 자동 넘김
- 매크로 방지용 창 자동 해제 기능 *(깜짝 미니게임! 같은 카드를 찾아라!)*
- 하트 소진시 자동대기 후 재진행
- ~~맵간 이동 자동화~~

---

## Usage

- 저전력으로 돌리기 위해 라즈베리파이4에서 scrcpy를 설치하여 안드로이드폰 미러링 형식으로 작동하기

---

## Caution

- 해상도 `1920x1080`에서 `전체화면`에서만 정상동작함
